### 서비스 개요
ESG Data API 서비스는 상장사들의 환경, 사회, 지배구조 데이터를 손쉽게 접근하고 활용할 수 있도록 하는 통합 솔루션입니다. 
기업의 ESG 성과를 실시간으로 조회하고 분석하여 지속가능한 투자 및 비즈니스 의사결정에 도움을 줍니다.
